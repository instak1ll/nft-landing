## Instalación y configuración

1. Clona o descarga este repositorio.
2. Instala las dependencias ejecutando el siguiente comando en la terminal:

`npm install`

## Uso

1. Ejecuta el servidor de desarrollo con el siguiente comando:

`npm run dev`
`npm run svgSprive`
